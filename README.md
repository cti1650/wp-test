# wp-test

## 目的

DockerでWordpressを構築する方法が載っている記事を見つけたため、
検証するためのリポジトリを作成しました。

## 環境構築

### Docker起動

```
docker-compose up -d
```

### Docker停止

```
docker-compose down --volumes
```
